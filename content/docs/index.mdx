## Three focused projects

Each project has a narrow responsibility:

- **Decimal** validates and calculates exact base-10 values without passing
  financial data through JavaScript floating-point numbers.
- **Mosaic** provides reusable trading UI behavior, accessible interaction,
  frontend validation, and presentation-ready state.
- **Relay** normalizes provider market data and moves it through caching,
  pub/sub, hydration, and real-time delivery.
- **Your application** connects these pieces and remains responsible for
  authentication, accounts, execution, permissions, compliance, and product
  policy.

Mosaic and Relay both use `@zagvar/decimal` for economic values. Mosaic does not
depend on Relay, and Relay does not depend on Mosaic.
